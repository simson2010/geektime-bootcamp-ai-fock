# Project Alpha 用户使用手册

## 快速开始

### 创建第一个 Ticket

1. 点击右上角的 **"+ 新建 Ticket"** 按钮
2. 输入 Ticket 标题（必填）
3. 可选：添加描述
4. 可选：选择标签
5. 点击 **"创建"** 按钮

### 管理 Ticket

- **编辑**: 点击 Ticket 卡片右上角的编辑图标
- **删除**: 点击 Ticket 卡片右上角的删除图标
- **完成/取消完成**: 点击 Ticket 左侧的复选框

## 功能详解

### 搜索功能

- 在顶部搜索框输入关键词
- 支持搜索 Ticket 标题
- 自动防抖，输入后自动搜索

**快捷键**: `Ctrl/Cmd + K` 聚焦搜索框

### 过滤功能

在左侧边栏可以：

1. **按状态过滤**:
   - 全部
   - 待完成
   - 已完成

2. **按标签过滤**:
   - 点击标签进行筛选
   - 可以同时选择多个标签
   - 点击 **"清除筛选"** 重置所有过滤

### 排序功能

在 Ticket 列表顶部：

- 选择排序字段：创建时间、更新时间、标题
- 点击排序按钮切换升序/降序

### 批量操作

1. 点击 **"批量操作"** 按钮
2. 选择要操作的 Tickets
3. 选择操作：
   - **批量完成**: 标记所有选中的 Tickets 为已完成
   - **批量删除**: 删除所有选中的 Tickets

### 标签管理

1. 点击 **"管理标签"** 按钮（桌面端在 Header，移动端在侧边栏）
2. **创建标签**:
   - 输入标签名称
   - 系统自动分配颜色（可手动修改）
   - 点击 **"创建"**
3. **删除标签**:
   - 点击标签右侧的删除图标
   - 确认删除

**注意**: 删除标签会从所有关联的 Tickets 中移除该标签。

## 键盘快捷键

- `Ctrl/Cmd + K`: 聚焦搜索框
- `N`: 创建新 Ticket（不在输入框时）
- `Esc`: 关闭对话框或侧边栏

## 移动端使用

- 点击左上角菜单图标打开侧边栏
- 搜索框在 Header 下方
- 标签管理在侧边栏底部

## 最佳实践

1. **使用标签分类**: 为不同类型的 Tickets 添加标签，便于后续筛选
2. **及时完成**: 完成后及时标记，保持列表整洁
3. **描述详细**: 添加详细描述有助于后续回顾
4. **批量操作**: 处理多个相似 Tickets 时使用批量操作提高效率

## 常见问题

### Q: 如何恢复误删的 Ticket？

A: 当前版本不支持恢复功能，删除前请确认。

### Q: 可以导出数据吗？

A: 当前版本不支持导出功能，后续版本会添加。

### Q: 支持多用户吗？

A: 当前版本为单用户模式，不支持多用户协作。

### Q: 数据存储在哪里？

A: 数据存储在 PostgreSQL 数据库中，确保定期备份。
